Project Description

StudySync is a comprehensive web application tailored for students. It features user authentication, note-taking capabilities, task management, and deadline tracking, all designed to help students manage their academic life more effectively.

Getting Started

Follow these instructions to get a copy of StudySync running on your local machine for development and testing purposes.

Prerequisites

Node.js
npm (Node Package Manager)
MongoDB (local or remote)

Installation

npm install

Configure environment:
Create a .env file in the root directory with necessary configurations:

DB_URI=mongodb://your_mongodb_uri
SESSION_SECRET=your_session_secret
PORT=3000

Running the Application

Start the server:

node server.js


Visit the application:
Open http://localhost:3001 in your web browser to access StudySync.
